> *「谁不想真正走进那些令自己魂牵梦绕的故事，亲身改变命运的走向呢？」*

**OmniStage（万象剧场）** 是那种——你点开之前以为是聊天软件，点开之后发现自己在**当造物主**的东西。

它不只想陪你唠嗑，它想给你一台 **「世界引擎」**：把那本让你意难平的小说搬进来也行，把你脑子里那套**力量体系乱飞、朝代换来换去**的架空脑洞落成能玩的舞台也行。总之：**你进场，故事开机。**

---

## 🧱 世界构建：别一上来就写论文，我们先盖房子

老实说，很多「酒馆」开局像布置寒假作业：世界观、势力表、人物小传……几万字没写完，人先麻了。

OmniStage 的打法更像个游戏：**先搭世界，再进本。**

- **世界熔炉（World Forge）≈ 你的「世界观编委会」**  
  想象后台坐着一排 AI：有人管年表，有人管地图，有人专门跟你抬杠「这条规则的代价是什么」。最后产出的是**能进系统、能迭代的世界宝典**，而不是聊天记录里那一坨「设定.txt 散落民间」。  
  你可以只扔一句「我想搞个靠吞诡异升级的架空大明」当种子，也可以把已经写好的设定集**嗖一下导入**——写完不是终点，**改版本才是乐趣**。

- **设定不是备忘录，是「下一把还会用到的宇宙」**  
  世界书、规则、时间线、角色卡会进版本。你今天改的每一条，都可能在明天的酒馆里**原样砸回你脸上**（褒义）。

- **角色别和世界离婚**  
  在世界里捏人或同步角色时，尽量让 TA 站在**同一套法则与历史**里说话——避免出现那种：人设帅到发光，但跟世界观毫无血缘关系的「OC 空降」。

**人话总结：**传统酒馆容易停在「我写了设定」；OmniStage 想把你带到「**我有一座会跟着剧情一起长胖、一起挨刀的城市**」。

---

## ⚡ 动态推演：世界不是壁纸，它会自己动

如果世界只是贴在提示词里的一堵长文墙，那你演得再用力，它也还是 **高清静态壁纸**——好看，但不吓人。

OmniStage 玩的是 **动态推演**：同一套设定，会在会话里被**推**出新局面，像有人偷偷按了「下一回合」。

- **台上不止你一个主角**  
  NPC 会互怼、会打配合、会抢戏；场面像一幕戏，而不是「请下一位同学发言」。

- **场面会自己往前走**  
  系统里有一套偏「导演 / DM」的编排思路：该加压加压，该收线收线。你不需要每次都自己当编剧——**偶尔让世界欺负你一下，也很好玩**。

- **骰子：专治「我台词都编好了怎么还能输」**  
  战斗、潜行、嘴炮这种不该被台词 100% 剧透结果的场合，交给 **掷骰 / 判定**。**骰子没落地前，大家都还是体面人。**

- **你说过的话，可能会回来找你**  
  分层记忆这类机制，让关键选择更容易跨回合埋雷、回收、打脸——长线 RP 的爽点，往往就是那句：**「你当初可不是这么说的。」**

**人话总结：**传统酒馆像 ping-pong；OmniStage 更想给你一种 **「这一格棋盘，自己翻过去了」** 的推背感。

## 🎭 和传统 AI 酒馆，差在哪？（一张表看懂）

别人很多产品在回答：**怎么聊得更像角色。**  
OmniStage 还想多问一句：**你们聊的，到底是不是同一个世界？这个世界，会不会被你玩变形？**

| 😵 你可能踩过的坑 | ✨ OmniStage 的取向 |
| --- | --- |
| 聊两集就忘设定，人设开始漂移 | 世界与角色走结构化资产，会话绑**世界版本**——少点「设定只活在第一页」的悲剧 |
| 世界薄得像 A4 纸，没有势力没有因果 | **世界构建**：法则、历史、组织、lore 进世界书，而不是一段超长 prompt 孤军奋战 |
| 剧情卡住，全靠玩家硬编 | **动态推演**：多角色同场 + 场面编排 + 掷骰，让局面**自己长刺** |
| NPC 只会围着你转，像轮流打卡 | 群像演出：NPC 也能互相接戏，**戏在台上**，不是只在你和模型之间 |
| 玩久了：我上周到底干了啥？ | **分层记忆**等机制，让关键事实更可能**活得比金鱼的记忆久** |

当然，OmniStage 不是许愿机——你总得告诉别人你想玩啥。但我们尽量做到：**别让你一个人扛下整个世界 + 全剧组 + 道具组。**

## 🚀 三分钟上手（新手村任务）

1. **搞个世界**：新建，或导入你的设定大礼包。  
2. **进世界熔炉**：先把你最在乎的几条补齐——规则、关键势力、主角团 / 宿敌雏形就行，**不用一次写到完美**（完美是后期 DLC）。  
3. **开酒馆会话**：记得**绑定世界版本**，不然容易变成「设定在 A 宇宙，人在 B 宇宙」。  
4. **演起来**：动作、台词、胡闹都行；卡关了试试自动推进，让世界踹你一脚。  
5. **设定不够用了？** 回炉增量补设定，再开新章——**长线连载，合法水字数**。

## 🛠️ 给开发者：先把引擎点着

```bash
git clone https://github.com/F0rJay/OmniStage.git
cd OmniStage
npm install
```

想**先不配模型密钥**也能点亮界面：复制 `apps/web/.env.example` → `apps/web/.env.local`，打开 `CW_CHAT_MOCK=1`，然后：

```bash
npm run dev:web
```

浏览器：`http://localhost:3000`

真机接模型、LiteLLM、Docker 等：`docs/litellm.md` 和 `docs/README.md`。

常用三连：`npm run dev:web` · `npm run build:web` · `npm run start:web`
